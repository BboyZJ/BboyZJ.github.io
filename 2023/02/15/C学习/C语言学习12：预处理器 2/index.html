C 预处理器

预处理器 不是 编译器的组成部分，是编译过程的一个单独的步骤。简言之，C预处理器是一个 文本替换工具，它会指示编译器在实际编译前完成所需的 预处理；我们把 C 预处理器(C Preprocessor) 简写成(CPP)

所有的预处理器都是以 # 开头，非空字符，为增加可读性，从第一列开始，下面列出了所有的预处理命令：
