typedef

C 语言提供了关键字 typedef ，可以用它为 类型 取一个新的名字。

例如：为无符号字符定义一个名称 UChar

typedef unsigned char UChar;

例如：为结构体定义一个名称 MyBook

typedef 和 #define 区别

  • typedef 仅限于为 类型 定义名称；#define 不仅可以为 类型 定义别买，也可以为 数值 定义

  • typedef由编译器 执行 解释；#define由编译器 预处理 处理
